    Описание

    Работа представляет API, написанное с использованием FastAPI, по теме "Разработка веб-приложения для поиска партнеров в путешествие". Используемая СУБД: PostgreSQL.

    Методы API

    Пользователи (Users)

    • GET /users/
      Описание: Получение списка пользователей.
      Параметры: Нет.

    • GET /users/me
      Описание: Получение данных текущего пользователя.
      Параметры: Нет.

    • GET /users/{user_id}
      Описание: Получение данных конкретного пользователя по его идентификатору.
      Параметры:

    • user_id (path, integer): Идентификатор пользователя.

    • PUT /users/{user_id}
      Описание: Обновление данных пользователя.
      Параметры:

    • user_id (path, integer): Идентификатор пользователя.
      Тело запроса: AccountUpdate (JSON).

    • DELETE /users/{user_id}
      Описание: Удаление пользователя.
      Параметры:

    • user_id (path, integer): Идентификатор пользователя.

    • POST /users/new
      Описание: Создание нового пользователя.
      Тело запроса: Account (JSON).

    • POST /users/login
      Описание: Авторизация пользователя.
      Параметры (Query):

    • username (string): Имя пользователя.

    • password (string): Пароль.

    • POST /users/register
      Описание: Регистрация нового пользователя.
      Тело запроса: Account (JSON).

    • POST /users/change-password
      Описание: Изменение пароля пользователя.
      Тело запроса: PasswordChange (JSON).

    Поездки (Trips)

    • GET /trips/
      Описание: Получение списка поездок.
      Параметры: Нет.

    • GET /trips/{trip_id}
      Описание: Получение данных поездки по её идентификатору.
      Параметры:

    • trip_id (path, integer): Идентификатор поездки.

    • PUT /trips/{trip_id}
      Описание: Обновление данных поездки.
      Параметры:

    • trip_id (path, integer): Идентификатор поездки.
      Тело запроса: Trip (JSON).

    • DELETE /trips/{trip_id}
      Описание: Удаление поездки.
      Параметры:

    • trip_id (path, integer): Идентификатор поездки.

    • POST /trips/new
      Описание: Создание новой поездки.
      Тело запроса: Trip (JSON).

    Попутчики (Companions)

    • GET /companions/
      Описание: Получение списка попутчиков для заданной поездки.
      Параметры (Query):

    • trip_id (integer): Идентификатор поездки.

    • POST /companions/
      Описание: Создание нового попутчика.
      Тело запроса: Companion (JSON).

    • PUT /companions/{companion_id}
      Описание: Обновление данных попутчика.
      Параметры:

    • companion_id (path, integer): Идентификатор попутчика.
      Тело запроса: Companion (JSON).

    • DELETE /companions/{companion_id}
      Описание: Удаление попутчика.
      Параметры:

    • companion_id (path, integer): Идентификатор попутчика.

    Маршруты (Itineraries)

    • GET /itineraries/{trip_id}
      Описание: Получение списка маршрутов для заданной поездки.
      Параметры:

    • trip_id (path, integer): Идентификатор поездки.

    • POST /itineraries/
      Описание: Создание нового маршрута.
      Тело запроса: Itinerary (JSON).

    • DELETE /itineraries/{trip_id}/{itinerary_id}
      Описание: Удаление маршрута.
      Параметры:

    • trip_id (path, integer): Идентификатор поездки.
    • itinerary_id (path, integer): Идентификатор маршрута.

    Сообщения (Messages)

    • GET /messages/{trip_id}
      Описание: Получение списка сообщений для заданной поездки.
      Параметры:

    • trip_id (path, integer): Идентификатор поездки.

    • POST /messages/
      Описание: Создание нового сообщения для поездки.
      Тело запроса: Message (JSON).

    Модели

    from enum import Enum
import datetime
from typing import Optional, List

from pydantic import BaseModel, EmailStr
from sqlmodel import SQLModel, Field, Relationship


class TripStatus(str, Enum):
    planned = "planned"
    in_progress = "in_progress"
    completed = "completed"
    canceled = "canceled"


class VehicleType(str, Enum):
    car = "car"
    motorcycle = "motorcycle"
    bicycle = "bicycle"
    scooter = "scooter"
    train = "train"
    bus = "bus"
    plane = "plane"
    boat = "boat"
    walking = "walking"


class CompanionStatus(str, Enum):
    confirmed = "confirmed"
    pending = "pending"
    declined = "declined"
    canceled = "canceled"


class AccountType(str, Enum):
    user = "user"
    admin = "admin"


class Account(SQLModel, table=True):
    id: Optional[int] = Field(default=None, primary_key=True)
    username: str
    first_name: str
    last_name: str
    email: EmailStr
    password: Optional[str] = Field(default=None, sa_column_kwargs={"nullable": True})
    description: Optional[str] = Field(default=None)
    created_at: datetime.datetime = Field(
        default_factory=datetime.datetime.utcnow, nullable=False
    )
    trips: List["Trip"] = Relationship(back_populates="initiator")
    role: AccountType = Field(default=AccountType.user)


class AccountUpdate(BaseModel):
    first_name: Optional[str]
    last_name: Optional[str]
    email: Optional[EmailStr]
    description: Optional[str]


class Trip(SQLModel, table=True):
    id: Optional[int] = Field(default=None, primary_key=True)
    initiator_id: Optional[int] = Field(default=None, foreign_key="account.id")
    start_date: Optional[datetime.datetime] = None
    end_date: Optional[datetime.datetime] = None
    vehicle: Optional[VehicleType] = None
    description: Optional[str] = None
    created_at: Optional[datetime.datetime] = None
    status: Optional[TripStatus] = None

    initiator: Optional[Account] = Relationship(back_populates="trips")
    companions: List["Companion"] = Relationship(back_populates="trip")
    messages: List["Message"] = Relationship(back_populates="trip")
    itineraries: List["Itinerary"] = Relationship(back_populates="trip")


class Companion(SQLModel, table=True):
    id: Optional[int] = Field(default=None, primary_key=True)
    trip_id: int = Field(foreign_key="trip.id")
    companion_id: int = Field(foreign_key="account.id")
    status: CompanionStatus
    created_at: datetime.datetime

    trip: Trip = Relationship(back_populates="companions")
    companion: Account = Relationship()


class Message(SQLModel, table=True):
    id: Optional[int] = Field(default=None, primary_key=True)
    trip_id: int = Field(foreign_key="trip.id")
    sender_id: int = Field(foreign_key="account.id")
    message: str
    created_at: datetime.datetime

    trip: Trip = Relationship(back_populates="messages")
    sender: Account = Relationship()


class Itinerary(SQLModel, table=True):
    id: Optional[int] = Field(default=None, primary_key=True)
    trip_id: int = Field(foreign_key="trip.id")
    stop_number: int
    location: str
    arrival_date: Optional[datetime.datetime] = None
    departure_date: Optional[datetime.datetime] = None
    created_at: datetime.datetime

    trip: Trip = Relationship(back_populates="itineraries")
